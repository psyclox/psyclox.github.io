🌟 About Cloud Crypt

Cloud Crypt is an innovative project designed by university students to provide an extra layer of **security and privacy** when storing files in cloud platforms like Google Drive. This project ensures **end-to-end encryption** using AES-256, making it impossible for unauthorized access.

📌 Why Cloud Crypt?

🔑 Data Security

Cloud storage is vulnerable to breaches. Cloud Crypt encrypts files before uploading, ensuring only authorized users can access them.

🚀 Ease of Use

The extension seamlessly integrates with Google Drive, allowing users to encrypt and decrypt files with a single click.

🔄 No Third-Party Access

Unlike traditional cloud security solutions, Cloud Crypt operates **locally**, ensuring encryption keys remain private.

⚙️ How It Works?

1️⃣ Encryption

The selected files are encrypted using **AES-256** before being uploaded to the cloud.

2️⃣ Secure Upload

The encrypted files are stored safely in **Google Drive** without exposing any data.

3️⃣ Decryption

Users can decrypt and retrieve their files anytime using their private password.
