🔐 Cloud Crypt

Advanced File Encryption for Secure Cloud Storage

🚀 Why Cloud Crypt?

Cloud Crypt is a **zero-knowledge encryption tool** that ensures **your files are encrypted before they reach the cloud**—keeping your data private even if your cloud provider is compromised.

🛠️ Key Features

  • 🔒 AES-256 Encryption with **PBKDF2 key derivation**
  • ☁️ Secure upload to **Google Drive**
  • 📜 Drag-and-drop support
  • ⚡ Optimized for fast encryption & decryption
  • 🎨 Interactive UI with smooth animations
