🔐 Cloud Crypt

Advanced File Encryption & Security

Developed by: Karthikeyan N, Mohamed Hashif H, Santosh M

Purpose: Cloud Crypt is an encryption tool that protects your files before uploading them to cloud storage. Using AES-256 encryption, it ensures your data remains safe even in case of a cloud breach.

🔍 How It Works

  • ✨ Select a file for encryption
  • 🔑 Enter a secure password
  • 🔐 File is encrypted before cloud upload
  • ☁️ Upload securely to Google Drive
  • 🔓 Download and decrypt anytime
