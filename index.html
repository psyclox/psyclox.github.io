Powerful Client-Side File Encryption for Cloud Storage

🚀 About Cloud Crypt

Cloud Crypt is a **zero-knowledge encryption tool** designed to **encrypt files before uploading** them to cloud storage. It ensures that even if the cloud provider is compromised, your data remains protected.

🛠️ Key Features

  • 🔒 AES-256 Encryption with **PBKDF2 key derivation**
  • ☁️ Secure upload to **Google Drive**
  • 🧑‍💻 Local client-side encryption (no server storage)
  • 📜 Drag-and-drop file support
  • ⚡ Optimized for fast encryption & decryption
  • 🎨 Interactive UI with smooth animations

🔍 How Cloud Crypt Works

1️⃣ File Selection

Users select the files they want to encrypt.

2️⃣ Password-Based Encryption

PBKDF2 derives a **secure AES-256 key** from a user-supplied password.

3️⃣ Secure File Upload

The encrypted file is uploaded to **Google Drive** via API.

4️⃣ Decryption on Download

Users retrieve files & decrypt them using the original password.
